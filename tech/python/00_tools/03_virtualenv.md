virtualenv
---

- Install virtualenv
    ```
    - python -m pip install --user virtualenv
    - python -m virtualenv --help
    ```

## User Guide

- Create the environment: `virtualenv env_name`
- Activate python environment: `source env_name/bin/activate`
- Confirm activation: `which python3`
